# 테마별 차이점 정리

## 가을 테마 (autumn)

### index.html
- ❌ 테마 전환 버튼 없음
- ✅ 가을 테마 배경 (낙엽 효과)
- ✅ 가을 테마 색상 (갈색/금색 계열)
- ✅ 챗봇 헤더 색상: 가을 테마 (#f4a460, #cd853f)

### list.html
- ❌ 겨울 이벤트 배너 없음
- ❌ 테마 전환 버튼 없음
- ❌ 룰렛 페이지 링크 없음
- ✅ 기본 UI만 표시

### 파일 목록
- `demo/themes/autumn/index.html`
- `demo/themes/autumn/list.html`

---

## 겨울 테마 (winter)

### index.html
- ❌ 테마 전환 버튼 없음
- ✅ 겨울 테마 배경 (눈 내리는 효과)
- ✅ 겨울 테마 색상 (보라색 계열)
- ✅ 챗봇 헤더 색상: 겨울 테마 (#667eea, #764ba2)

### list.html
- ✅ 겨울 이벤트 배너 표시
- ✅ 룰렛 페이지 링크 활성화
- ✅ 이벤트 페이지 접근 가능

### roulette.html
- ✅ 겨울 이벤트 룰렛 페이지

### 파일 목록
- `demo/themes/winter/index.html`
- `demo/themes/winter/list.html`
- `demo/themes/winter/roulette.html`

## 변경 사항 요약

### 가을 → 겨울 전환 시 추가되는 것
1. `index.html`: 가을 배경 → 겨울 배경
2. `list.html`: 배너 추가, 룰렛 링크 추가
3. `roulette.html`: 룰렛 페이지 추가

### 겨울 → 가을 전환 시 제거되는 것
1. `index.html`: 겨울 배경 → 가을 배경
2. `list.html`: 배너 제거, 룰렛 링크 제거
3. `roulette.html`: 제거 (또는 숨김)

---

## 이미지 태그 규칙

### 가을 테마 이미지
- 태그 형식: `autumn-YYYYMMDD-HHMMSS`
- 예: `autumn-20251120-172919`

### 겨울 테마 이미지
- 태그 형식: `winter-YYYYMMDD-HHMMSS`
- 예: `winter-20251120-170611`

## 검증 체크리스트

### 가을 테마 검증
- [ ] 로그인 화면에 테마 전환 버튼 없음
- [ ] 로그인 화면 배경: 가을 테마 (낙엽 효과)
- [ ] 목록 화면에 배너 없음
- [ ] 목록 화면에 테마 전환 버튼 없음
- [ ] 룰렛 페이지 접근 불가

### 겨울 테마 검증
- [ ] 로그인 화면에 테마 전환 버튼 없음
- [ ] 로그인 화면 배경: 겨울 테마 (눈 효과)
- [ ] 목록 화면 상단에 겨울 이벤트 배너 표시
- [ ] 배너에 '참여하기 →' 링크 있음
- [ ] 룰렛 페이지 접근 가능
